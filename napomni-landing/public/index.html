Политика конфиденциальности проекта Napomni

Дата вступления в силу: 03.04.2026

1. Общие положения

1.1. Настоящая Политика конфиденциальности определяет порядок обработки и защиты информации, которую проект Napomni получает от пользователей при использовании Telegram-бота, сайта, мобильных и иных цифровых сервисов проекта.

1.2. Используя проект Napomni, пользователь подтверждает, что ознакомился с настоящей Политикой и выражает согласие на обработку своих данных на условиях, изложенных в ней.

1.3. Если пользователь не согласен с условиями настоящей Политики, он должен прекратить использование проекта Napomni.

1.4. Проект Napomni предназначен для хранения, систематизации, поиска и напоминания информации, которую пользователь самостоятельно передаёт в сервис.

2. Какие данные могут обрабатываться

2.1. Проект Napomni может обрабатывать следующие данные пользователя:

  • имя, никнейм, username в Telegram;
  • номер телефона, если он был предоставлен пользователем;
  • текстовые сообщения, голосовые сообщения, фотографии, файлы, PDF-документы, заметки, списки, рецепты, напоминания, расписания и иные материалы, загруженные пользователем;
  • данные о событиях, задачах, датах, встречах и напоминаниях;
  • технические данные о взаимодействии с сервисом, включая дату, время, тип устройства, IP-адрес, сведения о браузере или приложении, системные журналы и служебную информацию;
  • иные данные, которые пользователь добровольно передаёт через сервис.

2.2. Пользователь понимает и соглашается с тем, что загружаемые им материалы могут содержать персональные данные как самого пользователя, так и третьих лиц.

2.3. Пользователь обязуется не передавать через проект Napomni данные третьих лиц без наличия у него законных оснований для такой передачи.

3. Цели обработки данных

3.1. Данные пользователя обрабатываются в следующих целях:

  • предоставление доступа к функциям проекта Napomni;
  • сохранение, систематизация и поиск загруженной пользователем информации;
  • создание напоминаний, уведомлений и подсказок по запросу пользователя;
  • улучшение качества работы сервиса;
  • обеспечение безопасности, предотвращение злоупотреблений и технических сбоев;
  • связь с пользователем по вопросам работы сервиса;
  • исполнение требований законодательства.

4. Правовые основания обработки

4.1. Проект Napomni обрабатывает данные на следующих основаниях:

  • согласие пользователя на обработку персональных данных;
  • необходимость исполнения соглашения с пользователем о предоставлении функционала сервиса;
  • необходимость соблюдения требований применимого законодательства.

5. Как используется информация пользователя

5.1. Информация, переданная пользователем, может использоваться для:

  • хранения в личном пространстве пользователя;
  • автоматической сортировки по категориям;
  • распознавания содержания текста, изображений, голосовых сообщений и документов;
  • формирования напоминаний и уведомлений;
  • поиска ранее сохранённых материалов по запросу пользователя;
  • улучшения алгоритмов работы сервиса и качества пользовательского опыта.

5.2. Проект Napomni не публикует загруженные пользователем материалы в открытом доступе без отдельного согласия пользователя.

6. Передача данных третьим лицам

6.1. Данные пользователя могут передаваться третьим лицам только в следующих случаях:

  • если это необходимо для работы проекта Napomni, включая размещение данных на серверах, использование облачной инфраструктуры, средств аналитики, уведомлений, распознавания речи, обработки документов и иных технологических сервисов;
  • если такая передача необходима для исполнения пользовательского запроса;
  • если передача обязательна по закону, по запросу суда, правоохранительных или иных уполномоченных органов;
  • если пользователь дал отдельное согласие на такую передачу.

6.2. При привлечении внешних сервисов и подрядчиков проект Napomni принимает разумные меры для защиты данных пользователя.

7. Хранение данных

7.1. Данные пользователя хранятся в течение срока, необходимого для достижения целей обработки, если более длительный срок не требуется по закону или не обусловлен характером сервиса.

7.2. Пользователь понимает, что смысл проекта Napomni заключается в долгосрочном хранении информации, которую пользователь передаёт для последующего поиска и напоминаний.

7.3. По запросу пользователя данные могут быть удалены, если иное не требуется по закону или не связано с необходимостью защиты законных интересов проекта Napomni.

8. Защита данных

8.1. Проект Napomni принимает необходимые организационные и технические меры для защиты данных пользователя от неправомерного доступа, изменения, раскрытия, блокирования, удаления, утраты или уничтожения.

8.2. Однако пользователь понимает, что ни один способ передачи информации через интернет и ни один способ электронного хранения не гарантирует абсолютную безопасность.

9. Права пользователя

9.1. Пользователь вправе:

  • получать сведения о своих данных и порядке их обработки;
  • требовать уточнения, обновления или исправления своих данных;
  • требовать удаления своих данных в предусмотренных законом случаях;
  • отзывать согласие на обработку персональных данных;
  • обращаться с вопросами и требованиями, связанными с обработкой данных.

10. Особо чувствительные данные

10.1. Пользователь самостоятельно принимает решение о передаче в проект Napomni документов и сведений повышенной чувствительности, включая медицинские документы, результаты анализов, сведения о здоровье, финансовые сведения и иные подобные материалы.

10.2. Передавая такие данные, пользователь подтверждает, что действует осознанно и добровольно.

10.3. Проект Napomni рекомендует пользователю не загружать сведения третьих лиц, медицинские документы третьих лиц и иные чувствительные данные без законных оснований и без согласия соответствующих лиц.

11. Использование Telegram и сторонних платформ

11.1. При использовании Telegram-бота обработка отдельных технических данных также может осуществляться платформой Telegram в соответствии с её собственными правилами и политикой конфиденциальности.

11.2. Проект Napomni не несёт ответственности за работу и правила обработки данных со стороны внешних платформ, если иное прямо не предусмотрено законом.

12. Изменение Политики

12.1. Проект Napomni вправе в любое время изменять настоящую Политику.

12.2. Актуальная версия Политики размещается на сайте проекта.

12.3. Продолжение использования проекта Napomni после публикации новой редакции Политики означает согласие пользователя с такой редакцией.
